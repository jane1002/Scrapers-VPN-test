## How to start
- npm i
- "npm run tx" start TX scraper 
- "npm run fl" start FL scraper
- "npm run ca" start CA scraper 


---
## TX

- Docket dataset:
    - docketNum
    - filings
    - utility 
    - description
    
- Filing dataset:
    - docketNum
    - itemNum 
    - fileStamp  
    - filingParty 
    - filingDescription            
    - download links(only PDF): array
